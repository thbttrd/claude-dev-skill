# plan-writing

> Plans the implementation of a **specific version** (V0, V1, ...) from project-tracking.json, producing a DAG of implementation plans organized into waves — where each wave delivers a vertical, testable, end-to-end slice of the app.

**Version:** 1.0.0 · **License:** MIT · **Part of:** [`claude-dev-skill`](../../README.md)

## Install

```
/plugin marketplace add github:thbttrd/claude-dev-skill
/plugin install plan-writing@claude-dev-skill
```
