# html-architect

> Generate architecture diagrams as hand-coded HTML + CSS + inline SVG — no D2, no TALA, no auto-layout engine — by applying TALA's software-architecture layout principles manually (symmetry, clusters, hierarchy, balanced ports, hub-centrality, square-ish aspect ratio).

**Version:** 1.3.0 · **License:** MIT · **Part of:** [`claude-dev-skill`](../../README.md)

## Install

```
/plugin marketplace add github:thbttrd/claude-dev-skill
/plugin install html-architect@claude-dev-skill
```
